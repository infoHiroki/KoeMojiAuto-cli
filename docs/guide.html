KoeMojiAutoの基本的な使い方

📦 インストール

1必要な環境を確認

  • Windows 10/11（64bit版）
  • Python 3.9以上
  • 8GB以上のメモリ推奨
  • 5GB以上の空き容量

2ダウンロード & インストール

GitHubからダウンロードして、インストールバッチを実行：

git clone https://github.com/infoHiroki/KoeMojiAuto-cli.git
cd KoeMojiAuto-cli
install.bat

3ショートカット作成

create_shortcut.bat

デスクトップにショートカットが作成されます。

🎯 基本的な使い方

1. 音声ファイルを配置

input/フォルダに音声・動画ファイルを入れます。

対応形式: MP3, WAV, M4A, FLAC, OGG, AAC, MP4, MOV, AVI

2. プログラムを起動

作成したショートカットをダブルクリック、またはコマンドで起動：

python koemoji.py

3. メニューから「1」を選択

メニューが表示されたら「1」を入力して文字起こしを開始します。

💡 ヒント: 初回起動時はAIモデルのダウンロードに10-30分かかります。

4. 結果を確認

output/フォルダに文字起こし結果（.txtファイル）が保存されます。

処理済みの音声ファイルはarchive/フォルダに移動されます。

⚙️ 設定のカスタマイズ

config.jsonを編集して動作をカスタマイズできます：

処理速度の調整

"whisper_model": "medium" // small/medium/large から選択
  • small: 最速、精度低
  • medium: バランス型（推奨）
  • large: 最高精度、処理遅い

高速化設定（CPU環境）

"compute_type": "int8" // CPUでの高速処理

GPU搭載PCの場合は"auto"に変更すると自動で最適化されます。

⚠️ 注意: 処理中にウィンドウを閉じると処理が中断されます。長時間の処理の場合は画面をそのままにしておいてください。
