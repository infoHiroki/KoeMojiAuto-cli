🚀 最新版をダウンロード

配布形式: ソースコード（GitHub）
動作環境: Windows 10/11
必要容量: 約5GB（モデルダウンロード後）
GitHubからダウンロード

※ GitHubのリリースページに移動します

💻 システム要件

必須要件

  • Windows 10/11（64ビット版）
  • Python 3.8以上
  • 8GB以上のRAM（16GB推奨）
  • 10GB以上の空きディスク容量
  • インターネット接続（初回モデルダウンロード時）

推奨スペック

  • Intel Core i5以上 または AMD Ryzen 5以上
  • 16GB以上のRAM
  • SSD（高速処理のため）
  • NVIDIA GPU（オプション：CUDA対応で高速化）

📝 インストール手順

1

ダウンロード

上記のボタンからGitHubのリリースページにアクセスし、最新版のZIPファイルをダウンロードします。

2

解凍

ダウンロードしたZIPファイルを任意の場所に解凍します。日本語パスでも問題ありません。

3

インストール実行

解凍したフォルダ内の install.bat をダブルクリックして実行します。必要なライブラリが自動的にインストールされます。

4

初回起動

install.bat の実行後、デスクトップに作成されたショートカットから起動できます。初回起動時は音声認識モデルのダウンロードに時間がかかる場合があります。

⚠️ 注意事項
  • 初回起動時、Whisperモデル（約1.5GB）の自動ダウンロードが行われます
  • ウイルス対策ソフトが警告を表示する場合がありますが、問題ありません
  • 管理者権限は不要です

📦 その他のダウンロード

開発者向け

ソースコードから実行する場合：

git clone https://github.com/infoHiroki/KoeMojiAuto-cli.git
cd KoeMojiAuto-cli
pip install -r requirements.txt
python koemoji_auto.py

過去のバージョン

過去のバージョンが必要な場合は、GitHubのリリースページからダウンロードできます。
